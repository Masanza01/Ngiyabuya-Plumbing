1. **Install Git** → [git-scm.com/download/win](https://git-scm.com/download/win)
2. **Create GitHub repo** → [github.com/new](https://github.com/new) — name it `ngiyabuya-plumbing`, set to **Public**
3. **Run the 6 git commands** shown in the guide
4. **Go to Vercel** → [vercel.com](https://vercel.com) → sign in with GitHub → import the repo → click Deploy

Vercel will host it for **free** and auto-redeploy every time you push a change. Your site will be live at `https://ngiyabuya-plumbing.vercel.app`.
